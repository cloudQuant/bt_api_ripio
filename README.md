# bt_api_ripio

Ripio exchange plugin for bt_api framework.

## Installation

```bash
pip install bt_api_ripio
```

## Usage

```python
from bt_api_ripio import RipioExchangeDataSpot, RipioRequestDataSpot
```

## Features

- REST API support
- HMAC-SHA256 authentication
- Symbol format: `BTC_USDT` (use underscore separator)
